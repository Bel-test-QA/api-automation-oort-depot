
## 🕷️ [BUG-101][API][starships] Security: BOLA Update- Юзер Б успешно модифицирует корабль Юзера А

**Ссылка на Jira:** [Открыть в Jira](https://atlassian.net)  
**Приоритет:** High / Medium / Low  
**Компонент:** API / Sandbox  

## 📝 Описание дефекта
Краткое описание проблемы, обнаруженной в ходе автоматизации.

## 👣 Шаги для воспроизведения (Steps to Reproduce)
1. Отправить `POST` запрос на эндпоинт `/api/v1/resource` с валидным телом.
2. Отправить точно такой же `POST` запрос повторно (создание дубликата).

> [TIP]
> Ожидаемый результат (Expected Result)
Сервер должен вернуть статус-код `400 Bad Request` и JSON с валидационной ошибкой: `{"error": "Resource already exists"}`.

> [CAUTION]
> ❌ Фактический результат (Actual Result)
Сервер падает со статус-кодом `500 Internal Server Error`.

## 💻 Логи / Тело ответа (Response)
```json
{
  "timestamp": "2026-07-02T12:00:00.000+00:00",
  "status": 500,
  "error": "Internal Server Error",
  "path": "/api/v1/resource"
}
```
